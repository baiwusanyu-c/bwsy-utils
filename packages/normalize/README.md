# @baiwusanyu/utils-normalize

## Function

### normalizePath

统一化路径，抹平win与 mac、linux 路径差异

| 参数   | 参数类型     | 默认值     | 说明      |
|------|----------|---------|---------|
| path | `string` | `-`     | 需要统一的路径 |

| 返回值 | 返回类型      | 说明 |
|-----|-----------|--|
| res | `string` | 统一化路径 |

### normalizeEllipsis

统一化文本过长，缩略展示

| 参数       | 参数类型     | 默认值  | 说明                    |
|----------|----------|------|-----------------------|
| str      | `string` | `""` | 需要统一的路径               |
| limitLen | `number` | `24` | 限制的文本字符数量，超过的部分会被省略显示 |

| 返回值 | 返回类型      | 说明 |
|-----|-----------|--|
| res | `string` | 缩略展示结果 |
